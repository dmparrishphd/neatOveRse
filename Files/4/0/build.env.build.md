build.env.build.R
=================

Attach an environment for building protopackages

Usage
-----

    source("~/build.env.build.R")[[1]]("~/Mirrors/")
    # where "~/Mirrors/" is the path to "GitHub/dmparrishphd/neatOveRse/", where the neatOveRse files are stored.
